# JavaScript Fundamentals - Learn by Day

A structured, hands-on JavaScript course for developers who want guided practice. Each day covers one concept with progressively challenging exercises.

## How to Use

1. Navigate to a day folder (e.g., `Day-1-Variables`)
2. Open a `.js` file
3. Read the instructions in the header comment
4. Run with Node: `node filename.js`
5. Solve the exercise, then check the solution below if needed

## Solutions

<details>
<summary>Day 4 - Functions Solutions</summary>

### 01-basic-function.js
```js
function greet() {
    console.log("Hello, World!");
}
greet();
```

### 02-function-parameters.js
```js
function greetPerson(name) {
    console.log(`Hello, ${name}!`);
}
greetPerson("Petar");
greetPerson("Alice");
```

### 03-return-values.js
```js
function add(a, b) {
    return a + b;
}

function multiply(a, b) {
    return a * b;
}

console.log(add(5, 3));      // 8
console.log(multiply(4, 7)); // 28
```

### 04-default-parameters.js
```js
function greetWithDefault(name = "Guest") {
    return `Welcome, ${name}!`;
}

console.log(greetWithDefault("Petar")); // Welcome, Petar!
console.log(greetWithDefault());         // Welcome, Guest!
```

### 05-arrow-functions.js
```js
const squareArrow = (x) => x * x;
const double = (x) => x * 2;
const isEven = (x) => x % 2 === 0;

console.log(squareArrow(5)); // 25
console.log(double(7));      // 14
console.log(isEven(4));      // true
```

### 06-function-scope.js
```js
function counter() {
    let count = 0;
    count++;
    return count;
}

// Each call returns 1 because count is recreated each time
console.log(counter()); // 1
console.log(counter()); // 1
```
